# JShell Commands. Java 9+ Required to run the following commands

### Open help
`/help`

### See the imports
`/imports`

### See the varialbes
`/var` or `/vars`

### Save 
`/save file.jshell`

### Open 
`/open file.jshell`

### List all the commands we have ran so far
`/list`

### Import 

### Edit. /edit opens an editor, while /edit #id, opens editor to modify that method.
`/edit`
